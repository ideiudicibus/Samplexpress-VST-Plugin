# Roadmap: Samplexpress

## Overview

Build a JUCE-based Classic Mode sample player — load a WAV/MP3, play it polyphonically with pitch-shifting across the keyboard, shape it with ADSR envelopes. Ships as VST3 plugin and standalone .exe.
